# POC 与 AVWAP 重新接受定义 v0.1

> 分线：C语义概念 / D风险管理 / H指标规则  
> 版本：v0.1  
> 状态：初版 / 已吸收部分既有资料 / 待进一步检索  
> 目的：明确“触碰、站回、重新接受、拒绝、回撤不破”的边界，避免把 POC/AVWAP 第一次接触误判为稳妥上车信号。  

---

## 1. 本文要解决的问题

本策略核心不是：

```text
价格碰到 POC 就买；
价格站上 AVWAP 就买；
价格回踩 VWAP 就买。
```

真正问题是：

```text
深洗后，价格是否重新接受了第一控盘区成本锚？
```

因此本文要定义：

```text
什么是触碰？
什么是站回？
什么是重新接受？
什么是拒绝？
什么是回撤不破？
POC 和 AVWAP 谁优先？
```

---

## 2. 当前核心定义

### 2.1 触碰

```text
价格到达或刺穿 POC / AVWAP 附近，但没有形成稳定收盘、停留或后续确认。
```

触碰只说明：

```text
价格到达成本锚附近。
```

不说明：

```text
市场已经接受该成本锚。
```

---

### 2.2 站回

```text
价格收盘重新回到 POC / AVWAP 上方，或重新进入第一控盘区价值区域内部。
```

站回比触碰强，但仍然不是完整接受。

站回只说明：

```text
短期价格重新穿越成本锚。
```

还需要观察：

```text
是否能停留；
是否回撤不破；
是否再次推进。
```

---

### 2.3 重新接受

[暂定定义]

```text
重新接受是指深洗后，价格重新回到第一控盘区价值区域或 POC / AVWAP 成本锚上方，并在后续一段结构中没有迅速跌回；第一次拉起后的回撤不再有效跌破关键成本锚，随后出现二次拉升确认。
```

更短定义：

```text
重新接受 = 站回成本锚 + 停留/换手 + 回撤不破 + 二次推进。
```

---

### 2.4 拒绝

```text
价格接近或短暂站上 POC / AVWAP 后，快速跌回下方，且后续多次尝试失败，使成本锚从潜在支撑变成压力。
```

拒绝意味着：

```text
成本区未被重新接受；
反抽可能只是诱多；
模型应降级为 observe_only 或 no_trade。
```

---

## 3. POC 的角色

POC 是：

```text
第一控盘区内部的核心成本点；
支撑/压力转换观察点；
深洗后价值区是否被重新接受的中枢锚。
```

POC 不是：

```text
买点；
止损点；
完整控盘区；
单独信号。
```

POC 重新接受需要观察：

```text
1. 是否重新进入第一控盘区价值区域；
2. 是否收盘站回 POC；
3. 站回后是否停留/换手；
4. 回撤是否不破 POC 或快速收回；
5. 二次拉升是否从 POC 上方展开。
```

---

## 4. AVWAP 的角色

AVWAP 是：

```text
从关键锚点开始计算的动态成本线。
```

在本策略中，主要 AVWAP 候选：

```text
AVWAP_ZoneStart：从第一控盘区起点开始；
AVWAP_ImpulseStart：从第一波启动K线开始；
AVWAP_Reclaim：从重新站回 POC 那根K线开始；
AVWAP_PullbackLow：从深洗低点开始。
```

优先级：

```text
1. AVWAP_ZoneStart：代表庄家初始控盘区平均成本；
2. AVWAP_ImpulseStart：代表第一波启动资金成本；
3. AVWAP_Reclaim：代表回收后新增资金成本。
```

---

## 5. 接受强度分级

### 5.1 弱接受

```text
价格收盘站上 POC / AVWAP >= 1 根K线。
```

含义：

```text
只代表短期站回，不足以作为稳妥确认。
```

---

### 5.2 中等接受

```text
价格收盘站上 POC / AVWAP >= 3 根K线，且没有快速跌回下方。
```

含义：

```text
成本锚短期从压力转为中性/潜在支撑。
```

---

### 5.3 强接受

```text
价格站上 POC / AVWAP 后，第一次拉起；随后回撤缩量不破关键成本锚；再出现二次拉升确认。
```

含义：

```text
这是本策略真正想研究的保守结构。
```

---

## 6. 回撤不破的定义

回撤不破不是绝对不能刺破，而是：

```text
不能有效跌破后持续停留在成本锚下方。
```

可以分为：

```text
强：回撤最低点不触及 POC/AVWAP；
中：刺破但收盘重新站回；
弱：跌破多根K线后才收回；
失效：跌破后 POC/AVWAP 变压力。
```

---

## 7. 常见误判

### 7.1 把触碰当接受

错误：

```text
价格碰到 POC，所以成本区回收。
```

修正：

```text
触碰只是到达，不是接受。
```

---

### 7.2 把一根K线站上当确认

错误：

```text
收盘站上 AVWAP，所以可以买。
```

修正：

```text
站上只代表穿越，必须看后续是否停留和回撤不破。
```

---

### 7.3 忽略价值区域

错误：

```text
只看 POC 一条线。
```

修正：

```text
应同时看是否重新进入第一控盘区价值区域，以及 VAH/VAL 边界。
```

---

## 8. 本版结论

当前核心定义：

```text
POC / AVWAP 重新接受不是触碰，也不是单根站上，而是站回成本锚后，能在成本区上方停留/换手，第一次拉起后回撤不破，并通过二次拉升确认。
```

策略执行含义：

```text
触碰 = 观察
站回 = 候选
停留 = 初步接受
回撤不破 = 关键确认
二次拉升 = 保守研究区
```
