# 第一控盘区成本中枢回收模型：策略词典 v0.2

> 目的：把个人中文交易语言转成统一、可审计、可编程、可搜索的概念词典。  
> 当前阶段：概念标准化 / 指标化准备。  
> 适用范围：MQL5 指标开发、样本标注、多机器人审查、后续回测规则设计。  

---

## 1. 策略总定义

### 1.1 中文名称

```text
第一控盘区成本中枢回收模型
```

### 1.2 完整定义

```text
第一控盘区形成后，价格经历第一波拉升和深回撤测试，随后重新接受第一控盘区 POC / AVWAP 成本锚，并在第一次拉起后的回撤中不破关键成本锚，最终通过二次拉升确认的保守观察结构。
```

### 1.3 英文映射

```text
First Control Zone Cost Basis Reclaim Model
Volume Profile POC Reclaim + Anchored VWAP Acceptance + Second Confirmation Model
```

### 1.4 不是以下模型

```text
不是 Fib 0.618 / 0.786 直接买入模型
不是 POC 第一次触碰买入模型
不是 VWAP 第一次回踩买入模型
不是左侧抄底模型
不是第一波追涨模型
不是单指标信号模型
```

## 2. 核心概念词典

| 中文术语 | 标准定义 | 英文/论坛关键词 | 在模型中的作用 |
|---|---|---|---|
| 第一控盘区 | 第一轮明显成交密集、价格停留、后续产生有效拉升的成本区 | First Control Zone, Initial Balance Zone, Accumulation Range | 模型根基；没有它，模型不成立 |
| 成本中枢 | 市场在一段区间内形成的主要成交/停留价格中心 | Cost Basis, Fair Value, Value Area | 判断价格是否重新接受该区 |
| POC | 指定区间内成交量或停留时间最大的价格水平 | Point of Control | 第一成本锚 |
| VAH | 价值区域上沿 | Value Area High | 控盘区上边界 / 突破参考 |
| VAL | 价值区域下沿 | Value Area Low | 控盘区下边界 / 风险边界 |
| AVWAP | 从特定锚点开始累计的成交量加权平均价 | Anchored VWAP | 第二成本锚 |
| 深回撤 | 第一波拉升后回撤到较深比例，通常观察 0.618-0.786 | Deep Retracement | 过滤浅回调，等待充分测试 |
| 重新接受 | 价格重新站回并稳定停留在关键成本区上方 | Acceptance, Reclaim | 判断成本区是否重新有效 |
| 第一次拉起 | 重新接受 POC/AVWAP 后的第一段反弹 | First Bounce, Initial Push | 不能直接视为确认 |
| 回撤不破 | 第一次拉起后的回测没有有效跌破 POC/AVWAP/VAL | Successful Retest | 保守确认核心 |
| 二次拉升确认 | 回撤不破后再次向上，并突破或接近第一次拉起高点 | Second Push Confirmation | 进入观察区前的最后确认 |
| 派发风险 | 第一波高位或反抽过程中出现卖压持续释放 | Distribution Risk | 风控否决条件 |
| 诱多反抽 | 成本区回收失败后的弱反抽或虚假突破 | Bull Trap, Failed Rally | 模型失效分类 |

## 3. 控盘区相关术语

### 3.1 第一控盘区

最低要求：

```text
1. 有可见成交密集或价格停留；
2. 后续从该区产生过明显拉升；
3. POC / HVN 能够被识别；
4. 区间不是随意画出的震荡框；
5. 该区后续能作为价格反应区。
```

建议量化方向：

```text
control_zone_bars >= 10
control_zone_range_atr <= 2.5 ATR
POC 位于区间内部，而非极端边缘
区间内成交量占观察窗口成交量比例 >= 30%
后续拉升幅度 >= 1.5 * control_zone_range
```

> 以上为初始建议阈值，后续需用样本校正。

---

### 3.2 POC

定义：

```text
指定 Profile 区间内成交量最大或价格停留次数最多的价格水平。
```

在模型中，POC 不等于买点，只是：

```text
成本锚
支撑/压力转换观察点
重新接受判断点
```

POC 有效性检查：

```text
POC 是否在第一控盘区内部
POC 是否被多次测试并产生反应
POC 是否在回撤后被重新站回
POC 是否从压力转为支撑
```

---

### 3.3 价值区域 VAH / VAL

定义：

```text
包含约 70% 交易活动的价格范围，上沿为 VAH，下沿为 VAL。
```

用途：

```text
VAH：判断是否离开价值区并向上扩展
VAL：判断深回撤是否破坏控盘区
POC：判断价值区中枢是否重新接受
```

## 4. AVWAP 相关术语

### 4.1 AVWAP 锚点候选

| 锚点名称 | 定义 | 用途 |
|---|---|---|
| AVWAP_ZoneStart | 从第一控盘区起点开始 | 观察完整控盘成本 |
| AVWAP_ImpulseStart | 从第一波启动K线开始 | 观察启动资金成本 |
| AVWAP_POCBuild | 从POC形成区间中点开始 | 观察中枢成本 |
| AVWAP_Reclaim | 从重新站回POC那根K线开始 | 观察回收后新增资金成本 |
| AVWAP_PullbackLow | 从深回撤低点开始 | 观察反弹资金成本 |

当前优先级：

```text
1. AVWAP_ZoneStart
2. AVWAP_ImpulseStart
3. AVWAP_Reclaim
```

---

### 4.2 AVWAP 接受

弱接受：

```text
价格收盘站上 AVWAP >= 1 根K线
```

中等接受：

```text
价格收盘站上 AVWAP >= 3 根K线，且回踩不破
```

强接受：

```text
价格站上 AVWAP 后，回踩缩量不破，随后再次放量上行
```

失效：

```text
站上后快速跌回
多次站上失败
AVWAP 变成压力
二次拉升前跌破且无法收回
```

## 5. 深回撤术语

### 5.1 深回撤区

观察区间：

```text
0.618 - 0.786
```

计算方式：

```text
retracement_ratio = (impulse_high - pullback_low) / (impulse_high - impulse_start)
```

注意：

```text
Fib 只用于描述回撤深度，不作为独立买点。
```

## 6. 二次确认术语

### 6.1 第一次拉起

定义：

```text
价格重新接近/站回 POC 或 AVWAP 后，出现第一段明显向上移动。
```

要求：

```text
不能直接作为入场确认
必须等待后续回撤测试
```

---

### 6.2 回撤不破

优先级从强到弱：

```text
不破 AVWAP + POC + VAL：强
不破 AVWAP + POC：中强
不破 POC：中等
只是不破前低：偏弱
```

---

### 6.3 二次拉升确认

最低要求：

```text
回撤后再次向上
突破或接近第一次拉起高点
成交量不明显衰竭
结构上出现 Higher Low 或 BOS
```

---

## 7. 风控术语

### 7.1 派发风险

风险信号：

```text
第一波高点巨量滞涨
连续上影线
价格不涨但成交量放大
反抽期间成交量放大但价格推进不足
POC 多次站回失败
AVWAP 成为压力
```

### 7.2 降级分类

| 情况 | 分类 |
|---|---|
| POC回收失败 | 弱反抽 |
| AVWAP回收失败 | 成本锚失效 |
| 二次拉升失败 | 诱多反抽 |
| 第一波派发严重 | 尾仓派发风险 |
| 回撤破坏VAL且无法收回 | 控盘区失效 |
