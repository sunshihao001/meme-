## 2. 策略本身需要先解决的 7 个问题

### 问题1：什么叫“庄家第一波控盘区域”？

需要明确：

```text
是成交密集区？
是低位横盘吸筹区？
是第一波启动前的成本区？
是链上筹码集中区？
是 Volume Profile 的 POC / HVN 区？
```

当前暂定：

```text
第一控盘区 = 第一波明显拉升前，价格停留和成交相对集中的庄家初始成本区。
```

---

### 问题2：第一波拉升的意义是什么？

第一波拉升不是为了追，而是为了证明：

```text
该成本区曾经具备控盘能力；
庄家曾经有能力把价格从该区拉开；
市场曾经被制造过注意力。
```

需要区分：

```text
健康第一波
拉盘出货第一波
纯噪音反弹
```

---

### 问题3：深洗和死亡盘如何区分？

meme 市场深回撤很常见，但不是所有深回撤都是洗盘。

需要区分：

```text
深洗盘：价格回撤深，但成本区后续能被重新接受。
死亡盘：价格回撤深，POC/AVWAP长期成为压力，反抽都被卖。
```

核心差异：

```text
深洗后能否重新回收成本锚。
```

---

### 问题4：后续起来时，什么叫“重新接受”？

不能简单理解为：

```text
价格碰到 POC
价格站上 VWAP 一根K线
价格涨了一点
```

更合理定义：

```text
价格重新站回第一控盘区成本中枢，并且在该区域上方产生持续成交；随后回撤时不再轻易跌回成本区下方。
```

---

### 问题5：什么叫“相对稳妥的上车位置”？

不是最低点，也不是第一根阳线。

本策略里的相对稳妥位置应是：

```text
重新接受 POC/AVWAP 之后，第一次拉起完成，再回撤不破关键成本锚，随后二次拉升确认后的区域。
```

换句话说：

```text
牺牲成本优势，换取结构确认。
```

---

### 问题6：如何区分二次启动和诱多反抽？

二次启动：

```text
回收成本锚
回撤不破
二次拉升有延续
卖压减弱
结构形成 higher low / BOS
```

诱多反抽：

```text
回收失败
POC/AVWAP变压力
反抽放量但推进弱
第二次拉升失败
高位继续卖出
```

---

### 问题7：策略到底属于什么类型？

最终定位：

```text
meme 市场庄家第一控盘区深洗后二次启动确认策略
```

更专业归类：

```text
Re-accumulation Reclaim Strategy
Market Maker Control Zone Retest Strategy
POC / AVWAP Cost Basis Reclaim Strategy
Washout Retest and Second Push Confirmation Strategy
```

## 3. 当前不做的事情

为了避免过早复杂化，当前不做：

```text
不定义精确买点
不定义止损止盈
不定义仓位
不做自动下单EA
不做参数优化
不做收益率回测
```

原因：

```text
策略语义还没完全打磨清楚，过早量化会把模糊概念伪装成精确规则。
```

## 5. 论坛搜索重点

因为 MQL5 / 国际论坛里不一定使用“庄家控盘”“洗盘”这些中文词，所以要用英文和俄文语义搜索。

中文概念对应：

```text
庄家控盘 = market maker control / smart money / operator / whale accumulation
洗盘 = washout / shakeout / spring / liquidity sweep
重新吸筹 = re-accumulation
重新接受成本区 = value area reclaim / POC reclaim / VWAP reclaim
二次启动 = second leg / markup continuation / second expansion
诱多反抽 = bull trap / failed rally / dead cat bounce
```
