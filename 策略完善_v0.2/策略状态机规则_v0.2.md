# 第一控盘区成本中枢回收模型：策略状态机规则 v0.2

> 目的：把策略路径拆成机器可判断的阶段，作为后续观察指标和标注器的核心规则。  
> 原则：只做阶段识别，不直接给交易建议。  

---

## 1. 状态总览

```text
S0：无结构
S1：第一控盘区形成
S2：第一波拉升完成
S3：深回撤测试
S4：重新接近 / 站回 POC
S5：重新接受 AVWAP
S6：第一次拉起
S7：回撤不破关键成本锚
S8：二次拉升确认
S9：稳定小段观察区
SX：失效 / 诱多 / 尾仓派发风险
```

---

## 2. 状态进入条件

### S0：无结构

进入条件：

```text
未识别到第一控盘区
或控盘区质量为 weak
或没有后续第一波拉升
```

输出：

```text
model_valid = false
allowed_mode = no_trade
```

---

### S1：第一控盘区形成

进入条件：

```text
control_zone_detected = true
control_zone_bars >= min_control_zone_bars
control_zone_range_atr <= max_control_zone_range_atr
poc_price 位于 control_zone_high / low 内部
```

建议初始阈值：

```text
min_control_zone_bars = 10
max_control_zone_range_atr = 2.5
```

状态意义：

```text
只说明存在可研究成本区，不代表可以买。
```

---

### S2：第一波拉升完成

从 S1 进入 S2 的条件：

```text
impulse_detected = true
impulse_high_price > control_zone_high
impulse_range >= 1.5 * control_zone_range
```

增强条件：

```text
impulse_volume_expansion >= 1.2
结构出现 higher high
```

风险提示：

```text
如果 impulse_quality = climax，需要提高派发风险评分。
```

---

### S3：深回撤测试

从 S2 进入 S3 的条件：

```text
pullback_detected = true
0.618 <= retracement_ratio <= 0.786
```

宽松观察区：

```text
0.5 <= retracement_ratio < 0.618：偏浅回撤
0.786 < retracement_ratio <= 0.886：偏深风险区
retracement_ratio > 0.886：结构破坏风险
```

S3 不能直接进入交易研究区，因为：

```text
深回撤只是测试，不是承接确认。
```

---

### S4：重新接近 / 站回 POC

进入条件：

```text
price touches or crosses poc_price
close_above_poc_bars >= 1
```

S4 分级：

```text
S4a：接近 POC，但未站回
S4b：单根收盘站回 POC
S4c：连续站上 POC >= 3 根
S4d：站上后回踩 POC 不破
```

S4d 才属于较有效回收。

---

### S5：重新接受 AVWAP

进入条件：

```text
avwap_enabled = true
price_above_avwap = true
close_above_avwap_bars >= 1
```

分级：

```text
S5a：单根站上 AVWAP
S5b：连续站上 AVWAP >= 3 根
S5c：回踩 AVWAP 不破
S5d：AVWAP + POC 同时被接受
```

强信号：

```text
S4d + S5d
```

---

### S6：第一次拉起

进入条件：

```text
first_bounce_detected = true
first_bounce_high > poc_price
first_bounce_high > avwap_value
```

增强条件：

```text
first_bounce_volume_ratio >= 1.0
first_bounce_high 接近或突破 VAH
```

注意：

```text
S6 仍不是确认，只是第一次反应。
```

---

### S7：回撤不破关键成本锚

从 S6 进入 S7 的条件：

```text
retest_after_bounce_detected = true
retest_holds_poc = true
或 retest_holds_avwap = true
```

分级：

```text
S7弱：只是不破前低
S7中：不破 POC
S7强：不破 POC + AVWAP
S7很强：不破 POC + AVWAP + VAL，且回撤缩量
```

失效条件：

```text
回撤有效跌破 POC 和 AVWAP，并在 N 根K线内无法收回
```

建议初始：

```text
reclaim_timeout_bars = 3
```

---

### S8：二次拉升确认

进入条件：

```text
second_push_detected = true
second_push_high >= first_bounce_high * tolerance
second_push_volume_ratio >= min_second_push_volume_ratio
```

建议：

```text
tolerance = 0.995
min_second_push_volume_ratio = 1.0
```

强确认：

```text
second_push_breaks_first_high = true
bos_confirmed = true
second_push_volume_ratio >= 1.2
```

---

### S9：稳定小段观察区

进入条件：

```text
S8 已确认
risk_level != high/extreme
distribution_score < max_distribution_score
```

建议初始：

```text
max_distribution_score = 60
```

S9 的意义：

```text
进入策略研究区。
不是保证盈利，只是满足保守结构。
```

---

### SX：失效 / 诱多 / 尾仓派发风险

任意阶段触发以下条件进入 SX：

```text
1. 控盘区被有效跌破且无法收回；
2. POC 多次回收失败；
3. AVWAP 变成压力；
4. 第一次拉起后回撤破掉关键成本锚；
5. 二次拉升失败；
6. 第一波高点派发严重；
7. 反抽过程中继续放量卖出；
8. distribution_score >= 70。
```

输出：

```text
model_valid = false
allowed_mode = no_trade
branch_type = bull_trap / distribution_risk / invalid
```


任意节点都可以进入：

```text
SX 失效 / 诱多 / 尾仓派发风险
```

---

## 4. 分支判断

### A. 派发后二次测试

条件：

```text
第一波有明显放量或派发迹象
深回撤后仍能重新接受 POC/AVWAP
二次拉升确认时卖压减弱
```

分类：

```text
branch_type = distribution_retest
allowed_mode = observe_only
```

---

### B. 第一次健康回撤承接

条件：

```text
第一波拉升健康
回撤缩量
POC/AVWAP重新接受顺畅
二次拉升有量能确认
```

分类：

```text
branch_type = healthy_pullback
allowed_mode = observe_only / small_size
```

---

### C. 诱多反抽 / 尾仓派发风险

条件：

```text
第一波派发严重
反抽继续卖
POC/AVWAP回收失败
二次拉升失败
```

分类：

```text
branch_type = bull_trap / distribution_risk
allowed_mode = no_trade
```

---

## 5. 初始评分建议

```text
control_zone_score: 0-20
poc_reclaim_score: 0-20
avwap_acceptance_score: 0-20
retest_score: 0-20
second_push_score: 0-20
risk_penalty: 0-100
```

综合分：

```text
confidence_score = control_zone_score + poc_reclaim_score + avwap_acceptance_score + retest_score + second_push_score - risk_penalty * 0.5
```

初始分类：

```text
confidence_score < 40：不成立
40-60：观察
60-75：结构成立但偏谨慎
75+：高质量结构，但仍需风控审查
```
