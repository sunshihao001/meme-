## 1. 资料核心内容

论坛资料共同强调：

```text
假突破是突破交易中最常见的问题之一。
市场接近关键位很强，不代表突破后能延续。
很多激进突破会迅速失败，形成 bull trap / bear trap。
```

---

## 2. 和本策略相关的概念

### 2.1 Fakeout

对应本策略：

```text
站回 POC/AVWAP 后快速跌回。
```

### 2.2 Bull Trap

对应本策略：

```text
深洗后第一次拉起吸引买盘，但后续无法回撤不破和二次推进。
```

### 2.3 Failed Rally

对应本策略：

```text
第一次反抽后无法延续，第二次拉升失败。
```

## 3. 可以吸收进知识库的内容

```text
强接近不等于强突破；
突破后收盘和后续停留更重要；
成交量放大但价格不推进，可能是派发；
假突破需要结合回撤、成交量、后续结构确认过滤。
```

对本策略转化为：

```text
站回成本锚后，如果价格不能停留，不能回撤不破，不能二次拉升，就是诱多反抽风险。
```
