## 1. 资料核心内容

Wyckoff 资料强调市场存在：

```text
Accumulation -> Markup -> Distribution -> Markdown
```

AMD 资料强调：

```text
Accumulation -> Manipulation -> Distribution
```

这两类资料都说明：

```text
真正有意义的结构不是单根K线，而是资金在某个区间完成吸筹/控盘后，再通过操纵或拉升离开该区间。
```

## 2. 和本策略相关的概念

### 2.1 Accumulation Range

对应本策略：

```text
第一控盘区的国际语义母体。
```

但 meme 市场中的第一控盘区往往是：

```text
压缩版、快速版、波动放大版 Accumulation Range。
```

---

### 2.2 Base Before Markup

对应本策略：

```text
第一波拉升前的底部/早期成本基地区。
```

它的意义不是买入，而是：

```text
确认庄家曾经从这里发动过第一波。
```

---

### 2.3 AMD 中的 Accumulation

对应本策略：

```text
早期吸筹/控盘区。
```

Manipulation 对应：

```text
深洗 / 扫流动性 / 假跌破。
```

Distribution 对应：

```text
第一波高位派发 / 尾仓派发 / 诱多反抽。
```

## 3. 可以吸收进知识库的内容

### 3.1 第一控盘区必须由后续拉升验证

单独的横盘不够。

必须有：

```text
从该区出发的第一波有效拉升。
```

这个拉升用于证明：

```text
该区曾经是有效成本区，而不是普通噪音区。
```

---

### 3.2 第一控盘区和派发区必须区分

Accumulation 和 Distribution 都可能表现为横盘和成交密集。

区分关键：

```text
位置不同：第一控盘区在第一波拉升前，派发区在第一波拉升后高位；
后续行为不同：控盘区后续拉升，派发区后续下跌或反抽被卖。
```

---

### 3.3 meme 第一控盘区不要求完整 Wyckoff 阶段

meme 可能没有传统完整五阶段。

更现实的是：

```text
短时间吸筹/控盘 -> 暴力第一波拉升 -> 深洗/操纵 -> 再判断是否重新吸筹。
```
