# 诱多反抽与二次启动区分 v0.1

> 分线：D风险管理 / B庄家周期 / C语义概念  
> 版本：v0.1  
> 状态：初版 / 已吸收部分既有资料 / 待进一步检索  
> 目的：区分 meme 深洗后重新起来时，到底是真二次启动，还是诱多反抽、尾仓派发或回收失败。  

---

## 1. 本文要解决的问题

在本策略中，价格经历：

```text
第一控盘区 -> 第一波拉升 -> 深洗 -> 重新接近成本区 -> POC/AVWAP站回
```

之后，最关键的问题变成：

```text
后续起来是真二次启动，还是诱多反抽？
```

如果误判：

```text
把诱多当二次启动 -> 接尾仓派发；
把二次启动当诱多 -> 错过保守确认段。
```

---

## 2. 当前定义

### 2.1 诱多反抽

```text
诱多反抽是指价格在深洗后看似重新回到第一控盘区或成本锚附近，但无法稳定接受 POC / AVWAP，也无法形成回撤不破和二次拉升确认；反抽过程中卖压持续出现，最终价格重新跌回成本区下方或继续下跌。
```

英文映射：

```text
bull trap
failed rally
failed reclaim
liquidity trap
exit liquidity
dead cat bounce
```

---

### 2.2 二次启动

```text
二次启动是指深洗后价格重新回收第一控盘区价值区域和 POC / AVWAP 成本锚，随后第一次拉起，回撤不破关键成本锚，再次拉升确认市场重新接受该成本区，并开始从成本区上方展开新的扩展结构。
```

英文映射：

```text
second leg
second push confirmation
markup continuation
re-accumulation breakout
SOS confirmation
LPS continuation
```

---

### 2.3 尾仓派发

```text
尾仓派发是指第一波高位已经明显派发，后续反抽不是重新吸筹，而是庄家利用重新回到成本区或热点恢复的机会继续卖出剩余筹码。
```

---

## 3. 阶段拆解

### 3.1 诱多反抽路径

```text
深洗
-> 反抽接近 POC/AVWAP
-> 短暂站上或触碰
-> 推进乏力
-> 回撤跌破成本锚
-> 第二次拉升失败
-> 重新跌回价值区下方
-> 进入死亡盘/尾仓派发风险
```

### 3.2 二次启动路径

```text
深洗
-> Sweep/Reclaim/Confirm
-> 重新进入 Value Area
-> 站回 POC/AVWAP
-> 第一次拉起
-> 回撤不破关键成本锚
-> 二次拉升突破或接近第一次拉起高点
-> 成本区从压力转支撑
-> 进入保守观察段
```

---

## 4. 诱多反抽表现

```text
1. POC/AVWAP 触碰后快速跌回；
2. 站回成本锚后停留时间短；
3. 第一次拉起成交放大但价格推进弱；
4. 回撤有效跌破成本锚；
5. 第二次拉升不能突破第一次拉起高点；
6. 上影线多，放量滞涨；
7. 反抽中主动卖压明显；
8. POC/AVWAP 从潜在支撑变成压力；
9. 热度短暂恢复但成交质量差；
10. 后续重新跌回价值区下方。
```

---

## 5. 二次启动表现

```text
1. 深洗后能重新进入第一控盘区价值区域；
2. POC / AVWAP 被重新站回；
3. 第一次拉起有结构推进；
4. 回撤缩量或卖压减弱；
5. 回撤不有效跌破关键成本锚；
6. 第二次拉升能突破或接近第一次拉起高点；
7. 结构形成 higher low；
8. 出现 BOS / SOS / LPS continuation；
9. 成本锚从压力转支撑；
10. 成交/流动性/关注度有恢复迹象。
```

---

## 6. 核心分界线

最重要分界不是：

```text
有没有反弹。
```

而是：

```text
反弹后回撤是否还能守住成本锚，并再次推进。
```

核心判断句：

```text
价格重新站回 POC / AVWAP 后，第一次拉起后的回撤是否不破关键成本锚，并在随后出现二次拉升确认？
```

---

## 7. 和 Order Block / Inducement / BOS 的关系

从 SMC 资料可吸收：

```text
诱导 Inducement 通常发生在真结构突破之前；
BOS 需要结合诱导和订单块反应才更可靠；
没有结构确认的突破容易是 liquidity trap。
```

对本策略转化为：

```text
第一次拉起可能只是诱导；
回撤不破和二次拉升更接近结构确认；
BOS / higher low / 成本锚支撑共振，才可升级为二次启动候选。
```

---

## 8. 本版结论

```text
诱多反抽 = 回收成本锚失败后的吸引买盘反抽。
二次启动 = 成本锚重新接受后，经过回撤不破和二次拉升确认的新一轮扩展结构。
```

对保守策略而言：

```text
第一次拉起不研究；
回撤不破开始重点观察；
二次拉升确认后才进入保守研究区。
```
